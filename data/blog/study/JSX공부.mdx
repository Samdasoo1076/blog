# JSX 란?

```JSX
const element = <h1>Hello, world!</h1>;
```

- 위 태그 문법은 문자열도, HTML도아닌 JSX(JavaScript + XML)라는 JavaScript에 XML을 추가하여 확장한 문법입니다.
- JSX는 React “엘리먼트(element)” 를 생성한다. React 엘리먼트는 브라우저 DOM 엘리먼트와 달리 일반 객체입니다.
- React는 JSX 사용이 필수가 아니지만, JS 코드 안에서 UI관련 작업을 할 수 있기 때문에 시각적으로 더 도움이 됩니다.
- JSX를 사용하면 React가 더욱 도움이 되는 에러 및 경고 메시지를 표시할 수 있게 해준다.
